# Sound Files

This directory contains notification sounds for Jarvis.

## gentle-chime.mp3

If this file is missing, NudgeManager will generate a gentle chime programmatically using Web Audio API.

To add a custom sound:
1. Find a royalty-free gentle chime or bell sound
2. Save it as `gentle-chime.mp3` in this directory
3. Recommended: 0.5-1 second duration, soft/pleasant tone
