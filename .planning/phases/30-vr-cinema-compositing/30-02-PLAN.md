Blender compositor node types used:
- CompositorNodeRLayers (Render Layers -- reads from a View Layer)
- CompositorNodeAlphaOver (Alpha Over -- composites with transparency)
- CompositorNodeMixRGB (Mix -- blend modes for layers)
- CompositorNodeGlare (Glare -- bloom/glow effects)
- CompositorNodeColorBalance (Color Balance -- grading)
- CompositorNodeComposite (Composite -- final output)
- CompositorNodeViewer (Viewer -- preview)

Blender View Layer system:
- Each View Layer renders a subset of the scene's collections
- scene.view_layers["ViewLayer"] is the default
- New view layers: scene.view_layers.new(name="...")
- Each view layer can include/exclude collections: view_layer.layer_collection.children[name].exclude = True/False
- Render Layers compositor node .layer property selects which View Layer to read

**Function 1: create_layer_collection(name, objects)**
Create a Blender Collection and add specified objects to it. This is the prerequisite for View Layer isolation -- each render pass renders only its collection.

- Create collection: `bpy.data.collections.new(name)`
- Link to scene: `bpy.context.scene.collection.children.link(collection)`
- Move each object (by name string) from its current collection into this one:
  - `obj = bpy.data.objects[obj_name]`
  - Unlink from all existing collections
  - Link to new collection: `collection.objects.link(obj)`
- Return the collection object
- Print JSON summary

**Function 2: setup_multi_layer(layer_configs)**
Create the complete multi-layer compositor from a list of layer configurations.

Args:
  `layer_configs` -- list of dicts, each with:
  - `name`: str -- layer name (e.g., "Fire", "Water", "Effects")
  - `collection`: str -- name of the collection this layer renders
  - `opacity`: float -- default opacity for this layer (0.0-1.0, default 1.0)
  - `bloom`: bool -- whether to add a Glare node after this layer (default False)
  - `bloom_threshold`: float -- glare threshold if bloom=True (default 0.8)

Implementation:
- `save_before_operate(tag="multi_layer_compositor")`
- Enable compositor: `scene.use_nodes = True`
- Clear existing nodes: `scene.node_tree.nodes.clear()`
- For each layer_config, create a View Layer:
  - `vl = scene.view_layers.new(name=config["name"])`
  - Exclude ALL collections from this view layer by default
  - Include ONLY the specified collection: iterate `vl.layer_collection.children`, set `exclude = True` for all except `config["collection"]`
  - Make the view layer render on transparent background: `vl.cycles.film_transparent = True` (so alpha-over works)
- Remove the default "ViewLayer" if it was not used: `scene.view_layers.remove(scene.view_layers["ViewLayer"])` -- but only if we created at least one other
- For each layer, create compositor nodes:
  - `CompositorNodeRLayers` with `.layer = config["name"]` (reads from the named view layer)
  - If bloom=True: `CompositorNodeGlare` with FOG_GLOW at config threshold
  - Position nodes in a grid: each layer gets a column, x offset by NODE_SPACING_X * column
- Chain layers together with Alpha Over nodes:
  - First layer's Image output is the base
  - Each subsequent layer: `CompositorNodeAlphaOver` -- Image (bottom) from previous chain, Image (top) from current layer
  - AlphaOver fac input = config["opacity"] (controls layer visibility)
- After the final Alpha Over, optionally add a Color Balance node (neutral by default -- users can adjust)
- Wire final output to both `CompositorNodeComposite` and `CompositorNodeViewer`
- Return dict with layer names, node names, and compositor summary
- Print JSON summary

**Function 3: add_render_pass(name, collection, opacity=1.0, bloom=False, bloom_threshold=0.8)**
Add a single new render pass to an existing multi-layer compositor (for incremental building).

- Create new View Layer for this pass
- Exclude all collections except the named one
- Set film_transparent = True
- Add new Render Layers node to compositor
- Find the last Alpha Over node in the chain (or the last Render Layers node if only one layer exists)
- Insert a new Alpha Over node: bottom = previous chain output, top = new layer
- Re-wire to Composite and Viewer outputs
- Return dict with the new layer info

**Function 4: composite_layers(output_name="composited", frame=None)**
Convenience function: render the multi-layer composite.

- Verify compositor has nodes (raise ValueError if empty)
- Import and call start_render or start_single_frame_render from async_render
- Return dict with output info

**Function 5: list_layers()**
Utility: list all View Layers and their collection assignments for debugging.

- Iterate scene.view_layers
- For each, list included/excluded collections
- Print JSON and return dict
