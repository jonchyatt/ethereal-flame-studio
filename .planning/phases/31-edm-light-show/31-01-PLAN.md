Create `blender/scripts/edm_light_template.py` following the fire_cinema_template.py four-function API pattern. The module must contain:

**Module-level setup (same pattern as fire_cinema_template.py):**
- `_SCRIPTS_DIR` with `__file__` fallback to `"C:/Users/jonch/Projects/ethereal-flame-studio/blender/scripts"`

- `REPO_ROOT`, `PRESETS_DIR` paths matching fire_cinema_template.py

**`_setup_edm_scene(preset)` -- Scene configuration:**
- Frame range, Cycles engine, resolution, samples, denoiser from quality preset (same as fire_cinema_template._setup_scene)
- `film_transparent = False` (opaque black background)
- World background: pure black, Strength=0.0 (NOT 1.0 -- scene starts in total darkness, effects are the only light)
- Motion blur enabled from preset
- Volumetric world: enable `world.cycles.volume_scatter_method = 'DISTANCE'` or add a Volume Scatter node to the world material for visible light shafts in spotlights. Use world material node tree: add Volume Scatter shader (density=0.02, anisotropy=0.3) connected to World Output Volume input. This makes spotlight cones visible as volumetric beams.

**`_create_lasers(preset)` -- Volumetric laser beams (EDM-01):**
- Create 4 Spot lights named `efs_edm_laser_0` through `efs_edm_laser_3`
- Position them in a row at y=-5, z=8, spread across x: x=-3, -1, 1, 3
- Point downward at angle (rotation_euler ~(0.8, 0, varied_z_rotation) so they fan out across the venue floor)
- Spot light settings per laser:
  - `energy`: 500 watts (bright enough to be visible as volumetric beams)
  - `spot_size`: 0.15 radians (~8.6 degrees, tight beam)
  - `spot_blend`: 0.1 (sharp edge with slight falloff)
  - `color`: Each laser a different color -- (1, 0, 0) red, (0, 1, 0) green, (0, 0.5, 1) blue, (1, 0, 1) magenta
  - `shadow_soft_size`: 0.01 (sharp shadows for laser look)
  - `use_shadow`: True
- Laser Z rotation will be keyframed by audio for sweep animation (Plan 02 handles this)
- Return list of laser objects

**`_create_led_grid(preset)` -- LED frequency display (EDM-02):**
- Create 8 emission cube meshes named `efs_edm_led_0` through `efs_edm_led_7`
- Arrange in a row (the "LED wall") at y=5, z=4, spread across x: x from -3.5 to 3.5 in equal spacing
- Each cube size: 0.8 x 0.8 x 0.8 (with 0.2 gap between them)
- Each LED gets its own Emission shader material named `efs_edm_led_mat_0` through `efs_edm_led_mat_7`
  - Emission color: gradient from warm to cool across the 8 columns (bass=red on left, treble=blue on right). Use: (1,0.2,0), (1,0.5,0), (1,0.8,0), (0.5,1,0), (0,1,0.5), (0,0.5,1), (0.2,0.2,1), (0.5,0,1)
  - Emission Strength: 10.0 (default, will be keyframed by audio)
  - Node tree: Emission shader -> Material Output Surface
- Return list of LED objects

**`_create_edm_camera()` -- Concert venue camera:**
- Position at (0, -10, 5) -- front of house looking at the stage
- Track To constraint targeting empty at (0, 0, 4) (center of laser/LED action)
- Focal length: 24mm (wide angle for venue feel)
- DOF enabled: f/4.0, focus distance 12m
- Name: `efs_edm_camera`, target empty: `efs_edm_target`

**`_create_fog_plane()` -- Ground fog for laser visibility:**
- Create a large cube (20x20x3) at (0, 0, 1.5) named `efs_edm_fog`
- Volume Scatter material: density=0.03, anisotropy=0.6 (forward-scattering for visible beams)
- This ensures laser beams have a medium to scatter through even if world volume is too thin
- `display_type = 'WIRE'` so it's not visually distracting in viewport

**`setup_edm_compositor()` -- Bloom and flare:**
- Render Layers -> Glare (FOG_GLOW, threshold=0.5, size=8, quality=HIGH) -> Color Balance (neutral, slight cool tint: offset=(0.95, 0.97, 1.0)) -> Composite + Viewer
- Lower bloom threshold than fire (0.5 vs 0.8) because laser/LED emission points should bloom heavily

**`create_edm_scene(quality="preview")` -- Main entry point:**
1. `load_quality_preset(quality)`
2. `save_before_operate("edm_light_scene_creation")`
3. Clear default scene
4. `_setup_edm_scene(preset)`
5. `_create_lasers(preset)`
6. `_create_led_grid(preset)`
7. `_create_edm_camera()`
8. `_create_fog_plane()`
9. `setup_edm_compositor()`
10. Print JSON summary (same pattern as fire_cinema_template)
11. Return summary dict

**`apply_edm_audio(audio_json_path, preset_name="edm_lights")` -- Audio wrapper:**
- Delegates to `keyframe_generator.apply_audio_keyframes(audio_json_path, preset_name=preset_name)`
- Same pattern as fire_cinema_template.apply_audio()

**`render_edm(output_name="edm_lights", frame=None)` -- Render wrapper:**
- Same pattern as fire_cinema_template.render_fire()
- Single frame or full animation via async_render

No bake function needed (no Mantaflow simulation -- all procedural geometry and lights).
