Create `src/components/ui/WidgetContainer.tsx` -- the reusable shell that wraps any widget content in a draggable, resizable floating panel.

The component uses react-rnd's `<Rnd>` component. Key behavior:

**Props interface:**

```typescript
interface WidgetContainerProps {
  /** Widget identifier */
  id: WidgetId;
  /** Display title shown in title bar */
  title: string;
  /** Current position */
  position: { x: number; y: number };
  /** Current size */
  size: { width: number; height: number };
  /** Current z-index */
  zIndex: number;
  /** Whether the widget is minimized (collapsed to title bar) */
  isMinimized: boolean;
  /** Minimum width constraint */
  minWidth: number;
  /** Minimum height constraint */
  minHeight: number;
  /** Called when user drags the widget to a new position */
  onDragStop: (id: WidgetId, position: { x: number; y: number }) => void;
  /** Called when user resizes the widget */
  onResizeStop: (id: WidgetId, size: { width: number; height: number }, position: { x: number; y: number }) => void;
  /** Called when user clicks anywhere on the widget (z-order) */
  onFocus: (id: WidgetId) => void;
  /** Called when user clicks minimize button */
  onMinimize: (id: WidgetId) => void;
  /** Called when user clicks close button */
  onClose: (id: WidgetId) => void;
  /** Widget content */
  children: React.ReactNode;
}
```

**Implementation details:**

1. **Rnd wrapper**: Use `<Rnd>` with these props:
   - `position={{ x: position.x, y: position.y }}`
   - `size={{ width: size.width, height: isMinimized ? WIDGET_TITLE_BAR_HEIGHT : size.height }}`
   - `minWidth={minWidth}`
   - `minHeight={isMinimized ? WIDGET_TITLE_BAR_HEIGHT : minHeight}`
   - `bounds="parent"` -- constrains to the parent WidgetLayer container (viewport)
   - `dragHandleClassName="widget-drag-handle"` -- only the title bar is draggable
   - `enableResizing={!isMinimized}` -- disable resize handles when minimized
   - `onDragStop={(e, d) => onDragStop(id, { x: d.x, y: d.y })}`
   - `onResizeStop={(e, direction, ref, delta, position) => onResizeStop(id, { width: parseInt(ref.style.width), height: parseInt(ref.style.height) }, position)}`
   - `onMouseDown={() => onFocus(id)}` -- bring to front on any click
   - `style={{ zIndex }}` -- dynamic z-index
   - `cancel=".widget-content"` -- do NOT let drag start from content area (prevents slider/input interference)

2. **Title bar**: A `div` with class `widget-drag-handle` containing:
   - Widget title (left-aligned, `text-white/80 text-sm font-medium`)
   - Minimize button: When minimized, shows a "restore" icon (expand); when expanded, shows a "minimize" icon (dash). Uses `onMinimize(id)`.
   - Close button: X icon. Uses `onClose(id)`.
   - Buttons: `text-white/60 hover:text-white` transition, 24x24 click targets with `pointer-events-auto`
   - Title bar background: `bg-white/10 hover:bg-white/15` with `cursor-grab active:cursor-grabbing`
   - Height: `WIDGET_TITLE_BAR_HEIGHT` (40px)

3. **Content area**: A `div` with class `widget-content` that:
   - Is hidden when `isMinimized` is true (`display: none` or conditional render)
   - Has `overflow-y: auto` for scrollable content
   - Has `flex: 1` to fill remaining space below title bar
   - Background: `bg-black/80 backdrop-blur-md`

4. **Outer styling**: The entire widget panel should have:
   - `border border-white/15 rounded-lg shadow-xl overflow-hidden`
   - `backdrop-blur-md` on the container
   - Background: `bg-black/80`
   - The panel should feel consistent with the existing ControlPanel styling (dark glass, white borders at 10-15% opacity)

5. **Important: Use 'use client' directive** at the top of the file since react-rnd uses DOM APIs.

6. **Import `WIDGET_TITLE_BAR_HEIGHT` from `@/types/widget`** for the minimize height constant.

**DO NOT:**
- Add animation/transitions on minimize (deferred, out of scope per REQUIREMENTS.md)
- Import or interact with any store (this component is pure props-driven)
- Add snap-to-edge behavior (WIDG-10 is deferred to v3.1)
