**Internal functions:**

1. `_clear_default_scene()` -- Same as fire_cinema_template.

2. `_setup_combo_scene(preset)` -- Configure Cycles for the combo scene:
   - Same base settings as water_template._setup_scene (Cycles, resolution from preset, denoiser, motion blur)
   - CRITICAL caustics settings: `scene.cycles.caustics_reflective = True`, `scene.cycles.caustics_refractive = True`
   - Increased bounces for water+fire interaction: `max_bounces = 16`, `glossy_bounces = 6`, `transmission_bounces = 16`, `transparent_max_bounces = 8`, `volume_bounces = 4`
   - `scene.render.film_transparent = False`
   - Do NOT set world background here (HDRI will handle it, or fallback to dark sky)

3. `_create_combo_camera(preset)` -- Camera optimized for fire-over-water framing:
   - Track target: `get_or_create_object("efs_combo_target", ...)` at `(0, 0, 1.5)` -- between water (0) and fire center (2.0)
   - Camera: `get_or_create_object("efs_combo_camera", ...)` at `(10, -8, 3)` -- further back than fire camera to show both water and fire
   - Focal length: `camera.data.lens = 50` (balanced framing for both fire and water)
   - DOF: `focus_distance = 12.8`, `aperture_fstop = 3.5`
   - Track To constraint
   - Set as active camera
   - Return camera

4. `_create_combo_lighting()` -- Lighting that works for both fire and water:
   - Sun light: `get_or_create_object("efs_combo_sun", ...)` at `(0, 0, 10)`. `SUN` type, energy 2.5, warm white `(1.0, 0.97, 0.92)`. Rotation: `(math.radians(55), 0, math.radians(25))` -- angled to create long reflections on water.
   - Rim light: `get_or_create_object("efs_combo_rim", ...)` at `(-5, 6, 5)`. `POINT` type, energy 40, cool blue `(0.7, 0.8, 1.0)`.
   - NOTE: The fire itself is the primary warm light source. The Mantaflow fire's emission lights the water via Cycles path tracing (this is what creates the caustics). No additional fire-specific lights needed beyond the sun and rim.
   - Return (sun, rim)

5. `_setup_combo_compositor()` -- Compositor balancing fire warmth and water coolness:
   - Enable compositing, clear nodes
   - Render Layers -> Glare (FOG_GLOW, threshold 0.85, size 7, quality HIGH, mix 0.0)
   - -> Color Balance (OFFSET_POWER_SLOPE): offset `(0.98, 0.97, 0.95)`, power `(1.0, 1.0, 1.0)`, slope `(1.02, 1.0, 0.98)` -- very subtle warmth without overpowering the water's cool tone
   - -> Composite + Viewer
   - Return dict

**Public API functions:**

1. `create_fire_water_scene(quality="preview", hdri_path=None)`:
   - Main entry point. Creates the complete fire-over-water scene.
   - Load quality preset: `load_quality_preset(quality)`
   - `save_before_operate(tag="fire_water_combo_creation")`
   - `_clear_default_scene()`
   - `_setup_combo_scene(preset)`
   - **Create water surface:** Call `_create_ocean(preset)` from water_template. The ocean is at `(0, 0, 0)` (the "ground plane").
   - **Create water material:** Call `_create_water_material(ocean)` from water_template. Glass BSDF with IOR 1.333.
   - **Create foam particles:** Call `_create_foam_particles(ocean)` from water_template.
   - **Create fire domain:** Call `_create_fire_domain(preset)` from fire_cinema_template. The domain is already at `(0, 0, 2.0)` with scale `(2.5, 2.5, 4)` -- fire floats above water.
   - **Adjust fire domain position for combo:** Set `domain.location = (0, 0, 2.5)` -- raise slightly so the fire base is clearly above the water surface. The flow emitter stays at `(0, 0, 0.3)` but we move it to `(0, 0, 0.8)` to be just above water.
   - **Create fire flow:** Call `_create_fire_flow()` from fire_cinema_template. Then adjust: `flow.location = (0, 0, 0.8)` (above water surface).
   - **Create fire material:** Call `_create_fire_material(domain)` from fire_cinema_template.
   - **Set fire cache directory:** `set_cache_directory(domain, "fire_water_combo")`
   - **Create combo camera:** `_create_combo_camera(preset)`
   - **Create combo lighting:** `_create_combo_lighting()`
   - **Setup HDRI (if provided):** If `hdri_path` is not None, call `setup_hdri(hdri_path, strength=0.8, rotation_z=0)` from world_template. The strength of 0.8 (not 1.0) avoids overpowering the fire's self-illumination. If no HDRI, set a dark blue-black world background: `(0.002, 0.003, 0.008, 1.0)` with strength 0.5.
   - **Setup compositor:** `_setup_combo_compositor()`
   - Build and print JSON summary (include all objects, their positions, caustics settings, HDRI status)
   - Return summary dict

2. `apply_combo_audio(audio_json_path, preset_name="fire_water_combo")`:
   - `return apply_audio_keyframes(audio_json_path, preset_name=preset_name)`

3. `bake_fire_water(clean_first=True)`:
   - Only the Mantaflow fire needs baking (ocean is procedural).

   - `start_bake(domain_name="efs_fire_domain", bake_type="ALL", clean_first=clean_first)`
   - Print: `[combo_fire_water] Fire bake started. Ocean is procedural (no bake needed).`

4. `render_fire_water(output_name="fire_water_combo", frame=None)`:
   - Same pattern as fire/water render functions.
   - Single frame: `start_single_frame_render(output_name, frame)`
   - Full animation: `start_render(output_name)`

**Object naming:** Fire objects keep their `efs_fire_*` names (from fire_cinema_template). Water objects keep their `efs_water_*` names (from water_template).
