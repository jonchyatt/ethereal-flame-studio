**Script header and imports:**

```python
"""
EFS Keyframe Generator -- reads audio analysis JSON + mapping preset,
inserts Blender keyframes on target objects/properties.

Usage via MCP execute_blender_code:
  import sys; sys.path.insert(0, 'C:/Users/jonch/Projects/ethereal-flame-studio/blender/scripts')
  from keyframe_generator import apply_audio_keyframes, load_preset, list_presets

  apply_audio_keyframes('C:/.../audio-analysis.json', 'meditation')
"""
import bpy
import json
import os
import shutil
from pathlib import Path

# Project paths
REPO_ROOT = Path("C:/Users/jonch/Projects/ethereal-flame-studio")
PRESETS_DIR = REPO_ROOT / "blender" / "presets"
```

**Core functions to implement:**

1. **`load_audio_json(filepath: str) -> dict`**
   - `json.load()` the audio export file
   - Validate required keys: `metadata`, `frames`, `summary`
   - Return the parsed dict
   - Raise `ValueError` with descriptive message if keys missing

2. **`resolve_feature_value(frame_data: dict, source_feature: str) -> float`**
   - Parse dot-notation source paths like `"bands.sub_bass"`, `"envelopes.bass"`, `"spectral_centroid"`, `"chromagram"` (returns max of 12 values for chromagram)
   - Navigate the frame dict to extract the value
   - Return float (0.0 if path not found, with warning print)

3. **`apply_mapping(obj, data_path: str, value: float, frame: int, curve_type: str)`**
   - Set the property value: use `exec()` with `f"obj.{data_path} = {value}"` for complex paths, or `obj.keyframe_insert(data_path=..., frame=frame)` for simple paths
   - For material node inputs, use `node.inputs[name].default_value = value` then `node.inputs[name].keyframe_insert("default_value", frame=frame)`
   - Insert keyframe at the given frame number
   - After insertion, set interpolation type on the keyframe's FCurve:
     - `"BEZIER"` with `handle_type = 'AUTO_CLAMPED'` for smooth envelopes
     - `"CONSTANT"` for onsets and beat triggers
     - `"LINEAR"` as fallback

4. **`apply_audio_keyframes(audio_json_path: str, preset_name: str = None, preset_path: str = None)`**
   - Main entry point. Loads audio JSON and preset.
   - If `preset_name` given, loads from `PRESETS_DIR / f"{preset_name}.json"`
   - If `preset_path` given, loads from that absolute path
   - If neither, uses `classification.suggested_preset` from the audio JSON
   - Calls `save_before_operate("keyframe_generation")` from scene_utils
   - For each mapping in the preset:
     - Find the target object in `bpy.data.objects[mapping["target_object"]]`
     - For each frame in audio JSON `frames` array:
       - Extract source value via `resolve_feature_value()`
       - Apply scale/offset/clamp: `value = clamp(source * scale + offset, clamp_min, clamp_max)`
       - Call `apply_mapping()` to set value and insert keyframe
   - Set scene frame range: `bpy.context.scene.frame_start = 1`, `frame_end = total_frames`
   - Set scene FPS from audio JSON metadata: `bpy.context.scene.render.fps = metadata["fps"]`
   - Print summary: number of mappings applied, total keyframes inserted, frame range

5. **`clamp(value: float, min_val: float, max_val: float) -> float`**
   - `return max(min_val, min(max_val, value))`

**Preset CRUD functions:**

6. **`list_presets() -> list[dict]`**
   - `PRESETS_DIR.glob("*.json")`, load each, return list of `{"name": ..., "description": ..., "path": ...}`

7. **`load_preset(name: str) -> dict`**
   - Load `PRESETS_DIR / f"{name}.json"`, parse, return dict
   - Raise `FileNotFoundError` with available presets listed if not found

8. **`save_preset(preset: dict, name: str = None)`**
   - If name not given, use `preset["name"].lower().replace(" ", "_")`
   - Write to `PRESETS_DIR / f"{name}.json"` with `json.dump(preset, indent=2)`
   - Create `PRESETS_DIR` directory if it doesn't exist (`os.makedirs(PRESETS_DIR, exist_ok=True)`)

9. **`delete_preset(name: str) -> bool`**
   - Remove `PRESETS_DIR / f"{name}.json"` if exists, return True
   - Return False if not found

10. **`create_preset(name: str, description: str, audio_style: str, mappings: list[dict]) -> dict`**
    - Build preset dict with full schema
    - Call `save_preset()` to persist
    - Return the preset dict

**Handling complex Blender data paths:**

The `apply_mapping` function must handle these 10 target property patterns:

```python
# Pattern 1: Fluid domain settings
# data_path: 'modifiers["Fluid"].domain_settings.flame_max_temp'
# Approach: obj.modifiers["Fluid"].domain_settings.flame_max_temp = value
#           then: obj.modifiers["Fluid"].domain_settings.keyframe_insert("flame_max_temp", frame=frame)

# Pattern 2: Fluid flow settings
# data_path: 'modifiers["Fluid"].flow_settings.fuel_amount'

# Pattern 3: Material node inputs (Principled Volume)
# data_path: 'active_material.node_tree.nodes["Principled Volume"].inputs["Density"].default_value'
# Must keyframe via: inputs["Density"].keyframe_insert("default_value", frame=frame)

# Pattern 4: Light data
# data_path: 'data.energy'
# Simple: obj.data.energy = value; obj.data.keyframe_insert("energy", frame=frame)

# Pattern 5: Light color (RGB)
# data_path: 'data.color'
# Special: map audio value to color temperature (0=red 1500K, 0.5=orange 3000K, 1.0=white 6500K)

# Pattern 6: Camera lens
# data_path: 'data.lens'
# Simple: obj.data.lens = value; obj.data.keyframe_insert("lens", frame=frame)

# Pattern 7: Particle emission (special -- count is not per-frame keyframeable)
# data_path: 'particle_systems[0].settings.count'
# Workaround: keyframe the emission start/end to create density variation

# Pattern 8: World background
# target_object: '__world__' (special sentinel)
# data_path: 'node_tree.nodes["Background"].inputs["Strength"].default_value'
# Access via bpy.data.worlds[0] instead of bpy.data.objects
```

Implement a `resolve_target(mapping)` helper that returns `(target_obj, property_path, keyframe_path)` tuple handling all 8 patterns. Use a mapping dict to route `data_path` prefixes to the correct Blender API calls.

**Important:** All frame indices in audio JSON are 0-based. Blender frames are 1-based. Add 1 when setting keyframes: `blender_frame = audio_frame["frame"] + 1`.
