<objective>
Create the production fire cinema template with multi-scale Mantaflow fire and configurable quality presets.

Purpose: This is the foundation script for all cinema-quality fire rendering. It supersedes fire_orb_poc.py by adding multi-scale detail (large billows + mid turbulence + fine wisps), a Principled Volume material with Blackbody temperature coloring, improved camera/lighting, and 4-tier quality presets (Draft through Ultra). The scene creation function produces a complete Mantaflow fire scene ready for audio keyframes and compositor effects.

Output: `fire_cinema_template.py` with `create_fire_scene(quality)` and `quality_presets.json` with Draft/Preview/Production/Ultra tiers.

Create `blender/scripts/fire_cinema_template.py` following the three-function API pattern from fire_orb_poc.py but with production-quality fire.

**Module docstring:** Explain this supersedes fire_orb_poc.py for production fire. Document the three-call MCP usage pattern (create_fire_scene -> bake_fire -> render_fire).

**sys.path setup:** Same __file__ / fallback pattern as fire_orb_poc.py (lines 48-54).

**Imports:** `bpy`, `json`, `math`, `sys`, `os`, `from pathlib import Path`. From scene_utils: `save_before_operate`, `get_or_create_object`, `set_cache_directory`, `full_scene_info`, `RENDERS_DIR`. Also define `PRESETS_DIR = REPO_ROOT / "blender" / "presets"` for quality preset loading.

**`load_quality_preset(name="preview")`:** Load from quality_presets.json. Returns dict. Raises FileNotFoundError with available presets if not found. Default is "preview" for a reasonable balance.

**`_clear_default_scene()`:** Same as fire_orb_poc.py -- select all, delete.

**`_setup_scene(preset)`:** Configure scene from quality preset dict:
- frame_start=1, frame_end=preset["frame_count"]
- render engine CYCLES
- resolution from preset (resolution_x, resolution_y, 100%)
- cycles.samples from preset
- denoiser OPENIMAGEDENOISE if preset["use_denoiser"]
- motion blur: scene.render.use_motion_blur = True, scene.render.motion_blur_shutter = preset["motion_blur_shutter"]
- film_transparent = False (black background for fire contrast)
- World background: pure black (same as POC)

**`_create_fire_domain(preset)`:** Multi-scale fire domain. Key differences from POC:
- resolution_max from preset (not hardcoded 64)
- cache_type = 'MODULAR'
- set_cache_directory(domain, "fire_cinema") -- different cache subdirectory from POC
- **Multi-scale parameters (the core improvement over POC):**
  - vorticity = 1.0 (was 0.5 in POC -- more turbulent large-scale motion)
  - use_dissolve_smoke = True, dissolve_speed = 15 (was 25 -- slower dissolve for longer flame trails)
  - use_noise = True, noise_scale = preset["noise_scale"] (from quality tier)
  - noise_strength = 1.0 (detail amplification)
  - flame_max_temp = 3.0 (controls peak flame brightness)
  - flame_smoke = 0.5 (some smoke mixed in for realism)
  - flame_vorticity = 1.5 (additional vortex detail in flames specifically)
  - burning_rate = 0.8 (fuel consumption speed)
- Domain scale: (2.5, 2.5, 4) for taller flame column (was 2,2,3)
- Domain location: (0, 0, 2.0) centered higher
- display_type = 'WIRE'

**`_create_fire_flow()`:** Enhanced flow emitter:
- Icosphere subdivisions=3, radius=0.4 (slightly larger than POC's 0.3)
- Location (0, 0, 0.3)
- flow_type = 'FIRE'
- flow_behavior = 'INFLOW'
- fuel_amount = 1.5 (was 1.0 -- more baseline fuel)
- temperature = 2.5 (was 2.0 -- hotter base)
- surface_distance = 1.5
- hide_render = True

**`_create_fire_material(domain)`:** Principled Volume with Blackbody on domain:
- Clear existing nodes
- ShaderNodeVolumePrincipled:
  - Color: (1.0, 0.35, 0.03, 1.0) -- deeper warm orange
  - Density: 8.0 (was 5.0 -- denser volume for more visible detail)
  - Blackbody Intensity: 1.5 (was 1.0 -- more pronounced temperature coloring)
  - Blackbody Tint: (1.0, 0.85, 0.7, 1.0) -- slight warm tint
- ShaderNodeOutputMaterial, Volume output connected

**`_create_camera()`:** Improved camera setup:
- Camera at (6, -6, 3.5) (slightly further back to frame taller flames)
- Track To constraint targeting empty at (0, 0, 2.0) (higher, matching domain center)
- Focal length 65mm (was 50mm -- tighter framing for more drama)
- Depth of field: camera.data.dof.use_dof = True, focus_distance = 8.5, aperture_fstop = 2.8 (cinematic shallow DOF)

**`_create_lighting()`:** Enhanced three-point lighting (not just one key light):
- efs_fire_key_light: Point light at (4, -3, 6), energy=150 watts, warm tint color=(1.0, 0.95, 0.85)
- efs_fire_rim_light: Point light at (-3, 4, 5), energy=50 watts, cool blue color=(0.7, 0.8, 1.0) -- rim/fill for depth
- efs_fire_ground_light: Area light at (0, 0, -0.5) facing up, energy=30, size=3.0 -- subtle ground bounce

**`create_fire_scene(quality="preview")`:** Main entry point.
1. Load quality preset via load_quality_preset(quality)
2. save_before_operate("fire_cinema_scene_creation")
3. _clear_default_scene()
4. _setup_scene(preset)
5. _create_fire_domain(preset)
6. _create_fire_flow()
7. _create_fire_material(domain)
8. _create_camera()
9. _create_lighting()
10. Print JSON summary (same pattern as fire_orb_poc.py) including quality preset name, all object names, scene settings
11. Return the summary dict

**`bake_fire(clean_first=True)`:** Wrapper that calls `from async_bake import start_bake; start_bake(domain_name="efs_fire_domain", bake_type="ALL", clean_first=clean_first)`.

**`render_fire(output_name="fire_cinema", frame=None)`:** Wrapper:
- If frame is not None: call `start_single_frame_render(output_name, frame=frame)` for single frame test
- If frame is None: call `start_render(output_name)` for full animation render
- Uses existing scene Cycles settings (set by _setup_scene)

**Object naming:** Reuse the same efs_ prefix names as fire_orb_poc.py (efs_fire_domain, efs_fire_flow, efs_fire_material, efs_fire_camera, efs_fire_key_light, efs_fire_target) PLUS new names for additional lights (efs_fire_rim_light, efs_fire_ground_light). This ensures the existing mapping presets (meditation.json, edm.json, cinematic.json) target the correct object names without modification.
