**Part A: Add setup_compositor() to fire_cinema_template.py**

Add a new function `setup_compositor()` that creates a compositor node tree for cinema fire. This should be added after the lighting function but before create_fire_scene(). The function:

1. Enable compositing: `scene.use_nodes = True`
2. Get the compositor node tree: `tree = scene.node_tree`
3. Clear existing nodes: `tree.nodes.clear()`
4. Create nodes:
   - **Render Layers** node (type `CompositorNodeRLayers`), location=(-300, 0)
   - **Glare** node (type `CompositorNodeGlare`), location=(200, 100):
     - glare_type = 'FOG_GLOW' (soft bloom, not streaks)
     - quality = 'HIGH'
     - mix = 0.0 (additive blend -- adds glow on top of original)
     - threshold = 0.8 (only bright fire areas bloom)
     - size = 7 (moderate glow radius)
   - **Color Balance** node (type `CompositorNodeColorBalance`), location=(500, 0):
     - correction_method = 'OFFSET_POWER_SLOPE' (ASC CDL)
     - offset = (1.0, 0.95, 0.88) -- slight warm tint in shadows
     - power = (1.0, 1.0, 1.0) -- neutral midtones (gamma)
     - slope = (1.05, 1.0, 0.95) -- warm highlights, cooler blues
   - **Composite** output node (type `CompositorNodeComposite`), location=(800, 0)
   - **Viewer** node (type `CompositorNodeViewer`), location=(800, -200) -- for background preview
5. Wire the nodes:
   - Render Layers "Image" -> Glare "Image"
   - Glare "Image" -> Color Balance "Image"
   - Color Balance "Image" -> Composite "Image"
   - Color Balance "Image" -> Viewer "Image"
6. Return a dict describing the compositor setup

Also add `setup_compositor()` call in `create_fire_scene()` -- call it as Step 8, after lighting but before the JSON summary. Update the return dict to include a "compositor" key with "bloom", "color_grading", "motion_blur" info.

Also add a new convenience function `apply_audio(audio_json_path, preset_name="fire_cinema")` that wraps `from keyframe_generator import apply_audio_keyframes; apply_audio_keyframes(audio_json_path, preset_name=preset_name)`. This gives a clean four-function API: create_fire_scene -> apply_audio -> bake_fire -> render_fire.

**Part B: Create fire_cinema.json mapping preset**

Create `blender/presets/fire_cinema.json` -- a fire-optimized mapping preset with 10+ mappings. This preset is designed specifically for the fire_cinema_template.py scene and maximizes the emergent complexity principle (8+ features to 8+ parameters simultaneously).

The preset must map:
1. **envelopes.bass -> efs_fire_flow fuel_amount** (bass drives fire intensity)
   - scale: 3.0, offset: 0.5, clamp: 0.2-3.5, curve: BEZIER
2. **onsets.bass -> efs_fire_flow fuel_amount** (bass hits create fuel spikes)
   - scale: 2.0, offset: 0.0, clamp: 0.0-5.0, curve: CONSTANT (sharp spike)
   - NOTE: This is a SECOND mapping to the same target. The keyframe generator overwrites per-frame, so the onset mapping should be applied AFTER the envelope mapping within the preset. Since they write to the same property, the onset value will override on onset frames (giving sharp spikes) while non-onset frames get the smooth envelope value. To achieve this, place the onset mapping AFTER the envelope mapping in the array.
3. **spectral_centroid -> efs_fire_domain flame_max_temp** (spectral centroid drives color temperature)
   - data_path: modifiers["Fluid"].domain_settings.flame_max_temp
   - scale: 4.0, offset: 1.0, clamp: 1.0-5.0, curve: BEZIER
4. **spectral_centroid -> efs_fire_material Blackbody Intensity** (centroid also drives material blackbody)
   - data_path: active_material.node_tree.nodes["Principled Volume"].inputs["Blackbody Intensity"].default_value
   - scale: 2.0, offset: 0.5, clamp: 0.3-2.5, curve: BEZIER
5. **envelopes.low_mid -> efs_fire_material Density** (mid frequencies drive smoke density)
   - data_path: active_material.node_tree.nodes["Principled Volume"].inputs["Density"].default_value
   - scale: 10.0, offset: 3.0, clamp: 2.0-12.0, curve: BEZIER
6. **rms_energy -> efs_fire_key_light energy** (overall energy drives key light)
   - data_path: data.energy, scale: 200.0, offset: 30.0, clamp: 20.0-300.0, curve: BEZIER
7. **spectral_centroid -> efs_fire_key_light color** (centroid drives light color temperature)
   - data_path: data.color, scale: 1.0, offset: 0.0, clamp: 0.0-1.0, curve: BEZIER
8. **envelopes.mid -> efs_fire_camera lens** (mid envelopes drive focal length breathing)
   - data_path: data.lens, scale: 20.0, offset: 55.0, clamp: 45.0-85.0, curve: BEZIER
9. **rms_energy -> __world__ background strength** (energy drives ambient glow)
   - data_path: node_tree.nodes["Background"].inputs["Strength"].default_value
   - scale: 0.3, offset: 0.02, clamp: 0.01-0.3, curve: BEZIER
10. **envelopes.sub_bass -> efs_fire_flow temperature** (sub-bass drives flow temperature)
    - data_path: modifiers["Fluid"].flow_settings.temperature
    - scale: 3.5, offset: 1.0, clamp: 0.5-4.0, curve: BEZIER
11. **envelopes.treble -> efs_fire_rim_light energy** (treble drives rim light for edge detail)
    - data_path: data.energy, scale: 80.0, offset: 20.0, clamp: 10.0-120.0, curve: BEZIER
