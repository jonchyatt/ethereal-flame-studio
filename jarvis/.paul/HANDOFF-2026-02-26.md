### Key Patterns Established
- **Spring easing:** `cubic-bezier(0.34, 1.56, 0.64, 1)` — used everywhere
- **Touch gestures:** Ref-based tracking (no React state during drag) for 60fps
- **CSS animations:** Keyframes via `<style>` tags in components, no libraries
- **Domain colors:** DOMAIN_COLORS lookup for safe Tailwind dynamic classes
- **Icons:** DomainIcon helper resolves string names → lucide components
