<objective>
## Goal
Fix tutorial so it runs all 4 Tier 1 lessons without stopping, and make the Continue chip visible even when chat is closed.

## Purpose
Jonathan's walkthrough found the tutorial stalls at the bills lesson and the Continue chip is invisible when chat is closed. Without a working end-to-end tutorial, the Jenny demo fails. This is the most critical fix.

## Root Causes (identified from code)
1. `bills-4-mark-paid` uses `personalStore.bills.some(b => b.status === "paid")` — with static bills, no bill can ever be marked paid → step PERMANENTLY fails → tutorial stuck forever. This is why the tutorial stops after habits.
2. Continue chip is rendered INSIDE ChatOverlay — if chat is closed during tutorial, the chip is hidden.
3. No step timeout — if any step verification fails indefinitely, tutorial stalls silently.

## Output
- Bills lesson: unachievable step replaced with achievable observation step
- Continue chip: floating button rendered at JarvisShell level, always visible during active tutorial
- Step timeout: 45s auto-advance with "Let's move on" message if step can't be verified
</objective>

    Create `TutorialContinueButton` as a local component in JarvisShell.tsx:
    ```tsx
    function TutorialContinueButton() {
      const engine = useTutorialEngineContext();
      const suggestedNext = useTutorialStore((s) => s.suggestedNext);

      // Show during active lesson OR between lessons with a suggestion
      const isActive = engine?.isActive ?? false;
      const hasSuggestion = !isActive && !!suggestedNext;

      if (!isActive && !hasSuggestion) return null;

      return (
        <button
          onClick={() => {
            if (isActive) engine?.skipStep();
            else if (suggestedNext) engine?.startLesson(suggestedNext);
          }}
          className="fixed bottom-[calc(5rem+env(safe-area-inset-bottom)+0.75rem)] left-1/2 -translate-x-1/2 z-[10002] px-5 py-2.5 bg-cyan-500 hover:bg-cyan-400 text-black text-sm font-semibold rounded-full shadow-lg shadow-cyan-500/30 transition-colors flex items-center gap-1.5 pointer-events-auto"
        >
          Continue <span>→</span>
        </button>
      );
    }
    ```
