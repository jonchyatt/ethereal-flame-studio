<objective>
## Goal
Elevate every existing E-04 component from "competent" to "premium" with consistent glassmorphism, hover elevation, active glows, and staggered entrance animations — matching the quality bar set by ChatOverlay and TodaySnapshot.

## Purpose
Before building more views (E-04-06+), the foundation must feel cohesive and delightful. Currently ~40% of interactive surfaces use flat `bg-zinc-900` instead of glass, no components have entrance animations, and hover states are basic color swaps without scale or shadow. This polish pass fixes the visual language before building on top of it.

## Output
- Card primitive gains `glass-interactive` variant (glass + hover elevation)
- All interactive cards across Home + Personal migrate to glass-interactive
- Shell chrome (Header, DomainRail, BottomTabBar) polished with glow + scale
- Staggered fade-in entrance animation applied to all list/grid components
</objective>
