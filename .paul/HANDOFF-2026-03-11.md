## What's In Progress

### THE CLAUDECODE FIX (Blocking Everything Else)

**Problem:** PM2 daemon inherits `CLAUDECODE=1` from Claude Code session. On Windows, `shell: true` in `start-web.js` spawns `cmd.exe` which re-injects the system env, overriding the `delete process.env.CLAUDECODE` fix.

**Solution (not yet applied):** Open a **native Windows cmd.exe or PowerShell** (NOT from Claude Code terminal), then:
```
cd C:\Users\jonch\Projects\ethereal-flame-studio
pm2 kill
pm2 start jarvis/ecosystem.config.js
pm2 start C:\Users\jonch\AppData\Local\cloudflared.exe --name jarvis-tunnel-cf -- tunnel run jarvis
pm2 save
```

**Alternative permanent fix:** Create a `.cmd` wrapper that does `set CLAUDECODE=` before `npm start`. Update ecosystem.config.js to use it.
